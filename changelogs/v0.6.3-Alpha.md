## News!
- Added `help` command to display a list of available commands, how to use them, and their descriptions.
- Bot now supports more characters as prefixes, default is still "!". You can change defualt prefix in .env file by modifiying the DEFAULT_PREFIX variable.
- Now you can set per user or per guild prefix via `settings` command, Currently supports:
    - "User Settings" to view and modify user-specific settings. (e.p. prefix )
    - "Server (General) Settings" to view and modify server-specific settings. (e.p. Server default prefix, Toggle On/Off Slash commands or message commands)
    - More settings will be added in future updates.
## Improvements!
- When sending unknown prefix command bot used to reply with a long message, now it will reply with a short message suggesting a close command to the one you typed.
- Unknown prefix command message can be disabled via "Server (General) Settings" by using /settings command.
